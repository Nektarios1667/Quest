.qlt - Quest Loot Table file

byte - table size

for each entry:
    byte - item ID - 1 = ActivePalantir, ...
    byte - percent chance
    byte - min amount
    byte - max amount

---------------------------------------
.qlp - Quest Loot Preset file

byte - width
byte - height

for each entry left->right, up->down:
    byte - itemID - 0 = empty, 1 = ActivePalantir, ...
    byte - amount
